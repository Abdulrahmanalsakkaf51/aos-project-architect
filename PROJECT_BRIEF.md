# AOS Project Architect — Project Brief

## Overview

AOS Project Architect is a Work & Productivity application that turns a real-world problem into a structured human–AI project.

## User Input

The user will provide:

- the problem;
- who is affected;
- the desired outcome;
- constraints.

## Generated Output

The future application will generate:

- a Project Brief;
- Partner Assignments;
- an Action Plan;
- Risks and Human Approvals;
- a reusable Knowledge Card.

## Human–AI Roles

- **Atlas** handles strategy, priorities, and milestones.
- **The Librarian** records assumptions, decisions, and reusable knowledge.
- **Guardian** identifies risks, privacy concerns, and decisions requiring human approval.
- **The Human Collaborator** approves important decisions and external actions.

## Core Principle

> “Every solution begins with a problem, every problem becomes a project, every project creates knowledge, and every piece of knowledge strengthens AOS.”
